# 🚀 DÉPLOIEMENT RAPIDE SUR VERCEL

## Méthode 1 : GitHub + Vercel (Automatique)

### Étape 1 : Préparer les fichiers
- Vérifiez que tous vos fichiers sont sur GitHub
- Le repo doit s'appeler `scouts-ndp` ou similaire

### Étape 2 : Connecter Vercel à GitHub
1. Allez sur **https://vercel.com/new**
2. Connectez-vous avec **GitHub**
3. Cliquez sur **"Import Git Repository"**
4. Sélectionnez votre repo `scouts-ndp`
5. Cliquez **"Import"** puis **"Deploy"**

✅ **Fini !** Votre site sera sur `https://scouts-ndp.vercel.app`

---

## Méthode 2 : Upload direct (Sans GitHub)

1. Allez sur **https://vercel.com/dashboard**
2. Connectez-vous avec votre email : `albinamani25@gmail.com`
3. Cliquez **"Add New..."** → **"Project"**
4. Cherchez l'option **"Upload"** ou importez depuis GitHub
